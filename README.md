
# https://4413frontend-production.up.railway.app/

# Features

- Catalog view
- Signin/signup
- View product
- Add to cart
- View order history
- Update cart
- Logout
- Filter catalog
- Checkout

- TypeScript - Fully typed API
- React application
- RTK query from Redux Toolkit to create query/mutation hooks
- Handle refetch 

# For local development

- Node v18.16.1 used

```
npm install
```

```
npm run dev
```
